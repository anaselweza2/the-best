Verify CPP Contributions and EI Premiums

Information

Throughout the Payroll Deductions Online Calculator (PDOC), additional information about terms is available through a link. If you select the link, a new browser window will open.

Verify CPP contributions
Verify EI premiums

This calculator uses the full CPP basic exemption ($3,500) for an employee and is valid only for full-year calculations.

For an employee who is pensionable for part of the year, such as a part-time employee, see Employers' Guide – Payroll Deductions and Remittances.

Step-by-Step Verification of CPP Contributions and EI Premiums

Welcome Page

The following calculations will be available on the Payroll Deductions Online Calculator page:

  • Salary
  • Commission
  • Pension
  • Verify CPP contributions and EI premiums

Note: You can only do one type of calculation at a time.

  1. Select the "Verify CPP contributions and EI premiums" radio button.
  2. Select the "Next" button to verify CPP contributions and EI premiums.

Step 1: General Information

When you select the "Verify CPP contributions and EI premiums" radio button, you will be directed to Step 1 of the verification of CPP contributions and EI premiums.

  1. The employee's name and/or the employer's name are optional fields.

    Note: If you enter the employee's name, it will automatically be included in the file name when you save the result at the end of the calculation. The application will accept a blank field.

  2. Select the province or territory of employment (normally where your business is located or where the employee reports for work). This is a required field.
  3. Enter the total CPP pensionable earnings of the employee for the year, up to the maximum pensionable earnings.

    Note: If the amount you enter is over the maximum pensionable earnings for that year, the system will automatically use the maximum pensionable earnings amount and a message will be displayed on the result screen.

  4. Enter the total EI insurable earnings of the employee for the year, up to the maximum insurable earnings.

    Note: If the amount you enter is over the maximum insurable earnings for that year, the system will automatically use the maximum insurable earnings and a message will be displayed on the result screen.

  5. Select the "Calculate" button to continue or the "Previous" button to return to the welcome page of the Payroll Deductions Online Calculator.

    Note: If the "Province or territory of employment" is not selected, an error message will display after the "Calculate" button is selected.

CPP Contributions and EI Premiums Calculated Result

The CPP contributions and EI premiums calculated result is based on the data you entered. You can use the "Print/Save Result" button to save and/or print the result.

To print and/or save the result in PDF format:

  1. Select "Print/Save Result" on each of the respective result screens. The default name of the file will be: Employee Name-CPP EI Calc-Annual-YYYY-MM-DD.pdf. If the employee's name, Joan Smith, is entered in Step 1, the file name will be Joan Smith-CPP EI Calc-Annual-2011-01-10.pdf.

To make changes to the current calculation:

  1. Select "Modify the Current Calculation" to go back to the previous input screen. Use this button if you want to modify the current calculation. All data will remain available.

To do another calculation:

  1. Select "Next Calculation" to go back to previous input screen. Use this button if you want to do a calculation for another employee. All tombstone data, except the employee's name, will remain available.

To select a different calculation type:

  1. Select "Start Over" to go back to the Welcome page. This will clear all the information entered in the previous steps.

To close the application:

  1. Select the "x" button of your browser window to close the application.
